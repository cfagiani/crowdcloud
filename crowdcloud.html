CrowdCloud for the Buffalo News

This site shows what's currently a hot topic of conversation in the comments section of The Buffalo News in the form of a word cloud.

How does this work?

The script downloadPosts.py will use the Disqus Posts API to download all the posts for a single forum for a given time range and write them to a file. The buildCloud.py script then can take that output file and build a JSON object containing the word frequencies. All words in the stopwords.txt file will be excluded from the cloud.

Where did this data come from?

The word cloud on this site came from The Buffalo News. The comments come from across all the articles published on the site. The words are presented in descending order of frequency (anything that appears to form a sentence is purely coincidental).
